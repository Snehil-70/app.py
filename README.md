# 🎤 Speech Sentiment Analysis

A simple app to **transcribe speech** and **analyze sentiment** using **OpenAI Whisper** and **Hugging Face Transformers**.  

You can **upload an audio file** or **record via mic**.

---

## **Try it Online**

[Open the App on Hugging Face Spaces](https://huggingface.co/spaces/Snehil006/speech-to-text)

---

## **Run Locally**

```bash
git clone https://github.com/yourgithubusername/speech_sentiment_app.git
cd speech_sentiment_app
pip install -r requirements.txt
python app.py
Then open the URL shown in your terminal (usually http://127.0.0.1:7860)
